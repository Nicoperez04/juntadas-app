# Resumen de uso de IA — Entrega 1

Durante la entrega 1 se usaron varias herramientas de IA con responsabilidades
distintas:

- **Claude / Claude Code CLI:** contextualizacion inicial, decisiones tecnicas,
  debugging estructurado y fixes de UX registrados en el Bloque Debugging.
- **Cursor Agent / Cursor IDE:** implementacion de bloques funcionales,
  generacion de codigo, fixes y consolidacion de conversaciones desde agent
  transcripts.
- **ChatGPT / Codex:** auditoria y correccion de documentacion IA, indice,
  estadisticas globales y registro de prompts/conversaciones.

La evidencia principal queda distribuida en:

- `ia/entrega-1/indice_ia.md`
- `ia/entrega-1/conversaciones/`
- `ia/entrega-1/prompts/`
- `ia/entrega-1/skills_instaladas.md`
- `ia/entrega-1/conversaciones/statistics.json`

`statistics.json` se considera un inventario global de las conversaciones y
transcripts documentados. No todos los registros conservan conteos exactos de
mensajes porque algunos fueron consolidados manualmente en Markdown.
