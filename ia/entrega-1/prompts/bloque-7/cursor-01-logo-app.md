# Bloque 7 — Marca: logo de la aplicacion

## Contexto

App Juntadas, React Native + Expo SDK 55 + TypeScript. El componente
`AppLogo` se usa en `WelcomeScreen` y `CompleteProfileScreen` como señal visual
de marca.

## Prompt

Ahora necesito que reemplacemos el icono que utiliza `AppLogo.tsx` con el
archivo `mobile/assets/logo-nobg.png` que agregue recien. Ese es el logo de la
aplicacion.

Esto tambien documentalo en la documentacion de IA bajo un nuevo Bloque 7 -
Marca.

## Restricciones asumidas

- Usar el asset local `mobile/assets/logo-nobg.png`.
- No crear ni modificar `.cursor/`.
- Mantener el tamaño definido por `theme.components.logoSize`.
- Documentar el cambio en `ia/entrega-1` como Bloque 7.
